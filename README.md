My project collection for 'simple pendulum'

1 file uses solve_ivp differential to calculate the angle, whereas the other file doesn't use solve_ivp differential($$\frac{d}{dt} \begin{bmatrix} \theta \\ \dot{\theta} \end{bmatrix} = \begin{bmatrix} \dot{\theta} \\ -\frac{g}{L} \sin(\theta) \end{bmatrix}$$) to calculate the angle. Instead it uses the Simple Harmonic Motion (SHM) formula:$$\theta(t) = \theta_0 \cos\left(\sqrt{\frac{g}{L}} \cdot t\right)$$
